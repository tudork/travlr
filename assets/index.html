flight API documentation

http://localhost:9000/

/flight/{from}/{to}/{departureDate}

/flight/{from}/{to}/{departureDate} get

get

Retrieve one-way trips for given origin, destination and date

get /flight/{from}/{to}/{departureDate}

Retrieve one-way trips for given origin, destination and date

  • Request
  • Response

URI Parameters

  • from: required (string)
  • to: required (string)
  • departureDate: required (string)

Query Parameters

  • Adult Count: (integer)

    Number of adults (default 1, max 9)

    Example:

    3
  • Child Count: (integer)

    Number of children (max 9)

    Example:

    1
  • Infant Count: (integer)

    Number of infants(max 9)

    Example:

    1
  • Infant In Lap: (integer)

    Number of infants in lap(max 9)

    Example:

    1
  • Infant In Seat: (integer)

    Number of infants in seat(max 9)

    Example:

    1
  • Senior Count: (integer)

    Number of seniors(max 9)

    Example:

    1
  • Max Stops: (integer)

    Maximmum number of stops (max 3)

  • Max Connection Time: (integer)

    Maximmum connection time (minutes)

    Example:

    70
  • Preferred Cabin: (one of None, Coach, Premium Coach, Business, First Class)

    Preferred Cabin (choose one)

    Example:

    None
  • Earliest Time: (integer)

    Earlies time of departure (HH:MM)

    Example:

    1000
  • Latest Time: (integer)

    Latest time of departure (HH:MM)

    Example:

    1900
  • Permitted Carrier: (string)

    List of permitted carriers (delimited by ',')

    Example:

    H1,AZ
  • Prohibited Carrier: (string)

    List of prohibited carriers (delimited by ',')

    Example:

    RO
  • Max Price: (string)

    Maximmum total price

    Example:

    RON500.00
  • Sale Country: (string)

    Country where the sale is to be made

    Example:

    RO
  • Refundable: (boolean)

    Is the ticket refundable?

    Example:

    true
  • Solutions: (integer)

    Number of returned solutions (default:20, max:500)

    Example:

    50

HTTP status code 200

Body

Type: application/json

Example:

{
 "data": {
  "respone": [{
   "pret": "EUR103.28",

   "FCO->OTP"

 },
